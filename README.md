# 桌面待办事项应用

一个美观易用的桌面待办事项应用，支持置顶显示和透明模式。

## 功能特点

- ✅ 动态壁纸模式，显示时钟和待办事项
- ✅ 窗口置顶，始终显示在最前面
- ✅ 透明模式，不遮挡其他内容
- ✅ 系统托盘，最小化到托盘
- ✅ 两个工作区，分别管理不同工作
- ✅ 本地存储，数据不丢失
- ✅ 美观的渐变紫色主题
- ✅ 动态粒子背景效果

## 安装和运行

### 方法一：直接运行（需要 Node.js）

**Windows 用户：**
1. 双击运行 `install.bat` 安装依赖
2. 双击运行 `start.bat` 启动应用

**或者使用命令行：**
1. 安装依赖：
```bash
npm install
```

2. 启动应用：
```bash
npm start
```

### 方法二：打包成独立应用

1. 安装依赖：
```bash
npm install
```

2. 打包应用：
```bash
npm run build
```

打包完成后，在 `dist` 文件夹中找到安装程序。

### 注意事项

- 需要先安装 Node.js（https://nodejs.org/）
- 如果没有图标文件，应用会使用默认图标
- 可以将 icon.svg 转换为 icon.png 和 icon.ico 以获得更好的显示效果

## 使用说明

### 窗口控制
- **置顶模式**：菜单 → 窗口 → 置顶（默认开启）
- **透明模式**：菜单 → 窗口 → 透明模式
- **最小化到托盘**：点击最小化按钮
- **从托盘恢复**：点击托盘图标

### 快捷操作
- 右键点击托盘图标可以快速切换置顶模式
- 双击托盘图标显示/隐藏窗口
